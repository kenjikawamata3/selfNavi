##環境構築手順
#1. Laravelプロジェクトを作成
>composer create-project --prefer-dist laravel/laravel ci
#2. サーバ起動
>cd ci
>php artisan serve
#3. install laravel/ui
>composer require laravel/ui
#4. install laravel/ui vue
>php artisan ui vue
#5. install npm 
>npm install
#6. install vue router
>npm install --save vue-router
※ERROR
#7. build
>npm run dev
